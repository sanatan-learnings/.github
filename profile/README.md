# Sanatan Learnings

**Sanatan Learnings** is an open-source effort grounded in **Hindu religion**—the scriptures, stories, and practices often gathered under **Sanātana Dharma**. We build careful, respectful tools for engaging sacred text **verse by verse**: readable Indic script and transliteration, more than one language where we can, and thoughtful additions such as search, pronunciation audio, and grounded question-answering when a project implements them. Everything here is free to fork, run locally, and improve with the community.

## Repositories

| Project | Purpose |
| -------- | -------- |
| [**hanuman-gpt**](https://github.com/sanatan-learnings/hanuman-gpt) | Hanuman-focused collections (e.g. Hanuman Chalisa, Sankat Mochan Hanumanashtak, Bajrang Baan, growing Sundar Kaand); bilingual study, audio, images, quizzes, search, and guidance. **Live:** [hanumanji.ai](https://hanumanji.ai/) · Apache-2.0 |
| [**shiva-gpt**](https://github.com/sanatan-learnings/shiva-gpt) | **Shiva Puran** in chapter and śloka order (381 verses); bilingual study, images, audio, quiz, and “Ask Shiva.” **Live:** [sanatan-learnings.github.io/shiva-gpt](https://sanatan-learnings.github.io/shiva-gpt/) · Apache-2.0 |
| [**krishna-gpt**](https://github.com/sanatan-learnings/krishna-gpt) | Early **Bhagavad Gītā** companion; all **701** ślokas scaffolded in canonical YAML, with rich commentary, media, and finished verse pages still mostly ahead. **Live:** [sanatan-learnings.github.io/krishna-gpt](https://sanatan-learnings.github.io/krishna-gpt/) · Apache-2.0 |
| [**sanatan-verse-sdk**](https://github.com/sanatan-learnings/sanatan-verse-sdk) | Python toolkit for canonical verse data and workflows: `verse-init`, `verse-generate`, embeddings, themes, and multi-collection projects. MIT |
| [**.github**](https://github.com/sanatan-learnings/.github) | Organization defaults and this profile page. |
