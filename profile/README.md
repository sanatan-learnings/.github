# Sanatan Learnings

**Sanatan Learnings** builds open, bilingual Jekyll sites for verse-level study—Devanagari and transliteration preserved, audio, search, and RAG-style guidance where each project enables it. [**Hanuman GPT**](https://github.com/sanatan-learnings/hanuman-gpt) ([hanumanji.ai](https://hanumanji.ai/)) covers Hanuman-focused collections; [**Shiva GPT**](https://github.com/sanatan-learnings/shiva-gpt) ([sanatan-learnings.github.io/shiva-gpt](https://sanatan-learnings.github.io/shiva-gpt/)) presents **Shiva Puran** in order (381 verses); [**Krishna GPT**](https://github.com/sanatan-learnings/krishna-gpt) ([sanatan-learnings.github.io/krishna-gpt](https://sanatan-learnings.github.io/krishna-gpt/)) is an early **Bhagavad Gītā** project: all **701** ślokas are scaffolded in canonical YAML, but full bilingual commentary, media, and browseable verse pages are only started—most of the work is still ahead. [**sanatan-verse-sdk**](https://github.com/sanatan-learnings/sanatan-verse-sdk) is the shared Python pipeline (`verse-init`, `verse-generate`, embeddings) behind parsing sources and generating text, images, and audio—fork, run locally, and contribute in whichever repo fits.
